# Scanner-documents

Cette application statique affiche des projets depuis un Google Sheet via Apps Script.

## Mise en ligne (GitHub Pages)

Le workflow GitHub Actions `.github/workflows/deploy-pages.yml` publie automatiquement le dépôt.

1. Poussez ce dépôt sur GitHub.
2. Dans **Settings → Pages**, choisissez **Source: GitHub Actions**.
3. Chaque push sur `main` ou `master` déclenchera le déploiement.

L’URL publique apparaîtra dans la section **Environments → github-pages** du dépôt.
